How to Get a Gemini API Key

If you don't know how to get a Gemini API key or don't even know what an API key is, don't worry. The instructions below will guide you through the process step-by-step. It takes less than 5 minutes to get your API key and it's free.

  1. Go to the Google AI Studio API Keys page.
  2. Sign in with your Google account if prompted.
  3. Click "Create API key" if you don't have one yet.
  4. Copy the generated API key.
  5. Paste the API key into the Gemini API Key field in FlashWave's settings and click "Save Settings".

That's it! Now you can start using FlashWave.

Note: Keep your API key private and do not share it with others.
